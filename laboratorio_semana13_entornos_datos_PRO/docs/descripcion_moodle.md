# Descripción lista para Moodle

## Nombre de la actividad

Semana 13: Laboratorio técnico avanzado - práctica guiada en el entorno de programación

## Tipo de actividad

Tarea / Assignment

## Modalidad

Individual o en parejas

## Descripción

En esta actividad el estudiante descargará un laboratorio en formato ZIP, lo abrirá en un entorno de programación como Visual Studio Code y ejecutará una práctica guiada con archivos CSV de gran tamaño. Durante el laboratorio deberá verificar el entorno, instalar dependencias, ejecutar scripts, explorar datos, generar reportes, corregir errores intencionales y comparar técnicamente el entorno usado según funcionalidad, compatibilidad y costo.

## Evidencia / entregable

Un informe final en PDF que incluya:

- capturas del entorno y la terminal;
- ejecución de comandos;
- resultados de scripts;
- archivos generados;
- depuración de errores;
- bitácora técnica;
- matriz de comparación técnica;
- conclusión argumentada.

## Criterios técnicos evaluados

- Navega paneles, carpetas, menús y terminal del entorno.
- Ejecuta comandos y scripts desde la terminal integrada.
- Verifica resultados y salidas del programa.
- Ajusta procedimientos mediante depuración básica.
- Interpreta archivos CSV con miles de registros.
- Compara el entorno según funcionalidad, compatibilidad y costo.
